# 用户手册

## 安装
- 下载并安装对应平台的安装包（Windows/macOS/Linux）。

## 界面与操作
- 左侧文件列表：新建、选择、删除。
- 右侧编辑区域：支持加粗、斜体、标题等基础格式。
- 搜索/替换：底部输入框输入“搜索”和“替换为”，点击“全部替换”。
- 夜间模式：右上角按钮切换。

### 表格（Word 风格面板）
- 插入表格：点击“插入表格”展开紧凑面板，在栅格中选择行列数即可插入。
- 面板功能：在同一面板中完成合并/拆分、水平/垂直对齐、边框样式（无/细/粗/虚线）、背景色（淡灰/黄色）以及“适应窗口/适应内容”自动调整。
- 响应式：面板在小屏幕上自动调整宽度与栅格尺寸。
- 右键菜单：在单元格处右键可进行上下文操作（选择整行/列、合并/拆分、行列增删、复制表格等）。

## 文件管理
- 新建：点击“新建”按钮。
- 保存：编辑完成后会在“应用库”内保存；后续支持另存为到本地路径。
- 删除：在列表中点击“删除”。

## 设置
- 主题偏好等设置将通过后端 Settings API 持久化。

## 同步（可选）
- 可在设置中启用同步并配置地址，推送/拉取变更。
