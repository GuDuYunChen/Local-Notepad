## 布局目标
- 固定整体宽度，工具按钮按功能分组，自动在多行排列；按钮文本保持单行，按钮不会因空间不足被挤压。

## 结构设计（示例）
```tsx
// Toolbar.tsx（示例结构）
export default function Toolbar() {
  return (
    <div className="toolbar-fixed">
      <div className="toolbar-stack">
        {/* 文本样式组 */}
        <div className="toolbar-group">
          <button className="btn primary">加粗</button>
          <button className="btn">斜体</button>
        </div>
        <span className="divider" />
        {/* 链接操作组 */}
        <div className="toolbar-group">
          <input className="input" placeholder="链接地址" />
          <button className="btn">链接</button>
        </div>
        <span className="divider" />
        {/* 历史操作组 */}
        <div className="toolbar-group">
          <button className="btn" title="Ctrl+Z">撤销</button>
          <button className="btn" title="Ctrl+Y / Ctrl+Shift+Z">重做</button>
        </div>
        {/* 第二行：文档结构组 */}
        <div className="toolbar-group">
          <button className="btn">H1</button>
          <button className="btn">H2</button>
          <button className="btn">H3</button>
          <button className="btn">P</button>
        </div>
        <span className="divider" />
        <div className="toolbar-group">
          <button className="btn">UL</button>
          <button className="btn">OL</button>
        </div>
      </div>
    </div>
  )
}
```

## 样式实现（示例）
```css
/* 固定布局容器：整体宽度固定，不响应 */
.toolbar-fixed {
  width: 960px; min-width: 960px; max-width: 960px;
}
/* 分组自动换行：按内容宽度决定每行包含的分组数量 */
.toolbar-stack {
  display: flex;
  flex-wrap: wrap;            /* 允许分组自动换行到下一行 */
  column-gap: 12px;           /* 组间水平间距 */
  row-gap: 12px;              /* 行与行之间的间距 */
  align-items: center;
}
/* 分组内控件：保持统一间距与对齐 */
.toolbar-group {
  display: inline-flex;
  gap: 8px;                   /* 按钮与输入框间距 */
  align-items: center;
  flex: 0 0 auto;             /* 禁止被压缩 */
}
/* 视觉分隔线 */
.divider { width: 1px; height: 24px; background: rgba(0,0,0,0.1); }
/* 按钮单行文本与最小宽度，防止挤压 */
.btn {
  white-space: nowrap;        /* 文本不换行 */
  min-width: 56px;            /* 按钮最小宽度 */
  padding: 6px 10px;
  border: 1px solid rgba(0,0,0,0.08);
  border-radius: 8px;
  background: transparent;
  color: var(--fg);
  flex: 0 0 auto;             /* 不随容器收缩 */
}
.btn.primary {
  background: var(--accent);
  color: #fff;
  border-color: var(--accent);
}
.btn:hover { background: rgba(126,91,239,.10); }
/* 输入框宽度规则：根据内容扩展但不低于最小值 */
.input {
  min-width: 220px;           /* 最小宽度保证可用性 */
  max-width: 40vw;            /* 在固定布局下仍可适当扩展 */
  white-space: nowrap;        /* 占位与内容单行显示 */
  padding: 6px 8px;
  border: 1px solid rgba(0,0,0,0.08);
  border-radius: 8px;
  background: transparent;
  color: var(--fg);
}
.input:focus { outline: none; border-color: var(--accent); box-shadow: 0 0 0 2px rgba(126,91,239,.25); }
```

## 规则匹配说明
- 按钮分组在多行展示：`toolbar-stack` 使用 `flex-wrap: wrap`，在固定宽度 960px 下自动决定每行可容纳的分组数量。
- 文本单行显示与按钮最小宽度：`white-space: nowrap` + `min-width: 56px`，防止内部文本折行与按钮被压缩。
- 行间与组间间距：`row-gap: 12px`、分组 `gap: 8px`；视觉分隔线 `divider` 统一高度 24px 保持对齐。
- 固定布局非响应式：父容器固定宽度（`width/min/max: 960px`），不随窗口缩放改变；但分组自动换行满足“每行数量按内容宽度自动调整”。
